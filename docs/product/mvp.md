# MVP Scope

This document defines the MVP scope for FoodSaver 0.1.0.

## Goal

Ship a first stable version focused on reducing household food waste through expiry date tracking.

## Included

- add food items
- track expiry dates
- highlight food expiring soon
- highlight food expiring today
- consume one quantity
- consume all remaining quantity

## Explicitly out of scope

- authentication
- multi-user support
- notifications
- barcode scanning
- bulk import
- categories
- analytics dashboard

## Release Criteria

The MVP is considered complete when:

- critical user journeys are covered by end-to-end tests
- accessibility checks pass
- deployment is stable
- API feature slices are covered by integration tests

## Consequences

+ Clear product focus
+ Faster iteration after release
+ Reduced maintenance scope

- Future features require explicit scope expansion
